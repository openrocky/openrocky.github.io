OpenRocky

面向 iOS、iPadOS 和 Android 的语音优先 AI Agent。用说的代替打字 —— Rocky 通过自然语音对话执行真实任务。

iOS 和 Android 均已开放测试。

iOS TestFlightAndroid 测试在 GitHub 查看
🎤

语音优先设计

语音是最核心的入口。在 iPhone、iPad 或 Android 设备上自然地和 Rocky 说话，文本只是补充 —— 这是真正的语音 Agent。

⚡

任务执行

Rocky 执行任务，不仅仅是对话。由 ROS 驱动 —— 组织会话、工具、记忆和产物的内部运行时。

📱

移动端原生

iOS/iPadOS 使用 SwiftUI 构建，Android 使用 Kotlin 构建。Rocky 从一开始就是为移动设备设计，而非桌面移植。

🔧

可扩展运行时

基于 ios_system 的本地执行层，支持 Python 和 WASM 扩展。构建在设备上运行的技能和工具。

🤖

多服务商

支持 OpenAI、Azure、Gemini、豆包等。服务商 / 账户 / 模型三层抽象，灵活切换。

🔓

开源
