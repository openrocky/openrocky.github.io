# 参与贡献

OpenRocky 是一个开源项目，欢迎社区贡献。

## 如何贡献

1. 在 [GitHub](https://github.com/openrocky/OpenRocky) 上 **Fork** 仓库
2. 在本地 **Clone** 你的 fork
3. **创建分支** 进行修改
4. **提交修改** 并测试
5. **提交 Pull Request**

## 开发环境

参阅 [快速开始](./getting-started.md) 了解如何搭建开发环境。

## 贡献方向

- **iOS App** — SwiftUI 功能、UI 改进、Bug 修复
- **运行时** — 执行层、技能/工具实现
- **文档** — 改进文档、添加指南、翻译
- **测试** — 单元测试、UI 测试、集成测试

## 交流

- [Discord](https://discord.gg/SvvsaDA4nE) — 社区讨论
- [Telegram](https://t.me/openrocky) — 更新与聊天
- [GitHub Issues](https://github.com/openrocky/OpenRocky/issues) — Bug 报告与功能建议

## 代码风格

- 遵循代码库中现有的 Swift 约定
- 新 UI 组件使用 SwiftUI
- 保持 commit 聚焦且描述清晰
